# ORIGIN Booking Rules

## Rule 1

A booking blocks the item for the selected dates.

---

## Rule 2

Only one active booking is allowed for a date range.

---

## Rule 3

Payment must complete before pickup.

---

## Rule 4

Pickup requires OTP verification.

---

## Rule 5

Return requires OTP verification.

---

## Rule 6

Owner must inspect the item before completing the booking.

---

## Rule 7

Late returns automatically trigger penalties.

---

## Rule 8

Cancelled bookings immediately release blocked dates.

---

## Rule 9

Every booking keeps a complete history.

Nothing is deleted.
